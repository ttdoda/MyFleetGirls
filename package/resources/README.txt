これはなに？

MyFleetGirls の為のクライアントアプリケーションです。
詳細は以下のサイトを参照。基本的な使い方も載っています。
  http://myfleet.iwmt.org/entire/about

細かい設定方法(Windows 用)：
 1. proxy の設定
    各種ブラウザに proxy 設定を行います。
    自動プロキシ設定スクリプトを用意していますので、以下のアドレスを設定するだけです。
      http://myfleet.iwmt.org/assets/proxy.pac
    IE および IE コンポーネント系の艦これ専用ブラウザーをしようしている方は同
    梱の IE_PROXY.REG をダブルクリックすると楽に設定出来ます。

 2. 起動
    同梱の myfleetgirls.bat をダブルクリックして起動してください。
    その他の方法での利用は基本的にサポート外になります。
    起動中はコマンドプロンプトが表示されますが、艦これプレイ中は閉じないように
    してください。
    コマンドプロンプトに表示されている情報は通常の利用にはあまり影響は有りませ
    んが、サポート等で必要になる場合があります。
